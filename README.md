# Fancy Counter

A stylish and interactive counter web application built with HTML, CSS, and JavaScript.

## Features

- Increase and decrease counter values
- Reset functionality
- Keyboard controls (Arrow Up/Down to change, other keys reset)
- Visual limits (0-5 range)
- Responsive design
- Modern UI with gradients and icons

## Demo

View the live demo: [https://mirhadidev.github.io/fancy-counter-update3/](https://mirhadidev.github.io/fancy-counter-update3/)

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6+)
- Font Awesome Icons
- Google Fonts
